# Cobaltz Theme for Obsidian

A beautiful dark and light theme based on Cobalt2 with refined colors, optimized for focused writing and note-taking in Obsidian.

## Installation

### From Obsidian Community Themes

1. Open Obsidian Settings
2. Go to Appearance → Themes
3. Click "Manage" and search for "Cobaltz"
4. Click Install and Use

### Manual Installation

1. Download `theme.css` and `manifest.json`
2. Create a folder called `Cobaltz` in your vault's `.obsidian/themes/` directory
3. Copy both files into the `Cobaltz` folder
4. Open Obsidian Settings → Appearance → Themes
5. Select "Cobaltz" from the dropdown

## Features

- Deep blue background for reduced eye strain (dark mode)
- Soft off-white background (light mode)
- Golden yellow headings with proper weight hierarchy
- Vibrant syntax highlighting for code blocks
- Styled callouts (info, warning, error, success)
- Custom graph view colors
- Matching status bar styling
- Comfortable selection highlighting
- Auto-switches between dark/light based on system preference
